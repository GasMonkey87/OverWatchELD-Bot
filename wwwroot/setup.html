OverWatch ELD First-Time Setup

Step-by-step guide for login, VTC connection, telemetry, and key features.
Click each topic below to expand the instructions. Start at the top and work your way down.
1. Download and install OverWatch ELD
  1. Go to the downloads page.
  2. Click Download Installer.
  3. Run the installer after it finishes downloading.
  4. Open OverWatch ELD from your desktop or Start Menu.
Required first step
2. First-time login
  1. Open OverWatch ELD.
  2. On the login screen, enter your driver name or profile information.
  3. If your VTC uses Discord linking, keep the app open and continue to the Discord setup step.
  4. After login, you should see the main dashboard with duty clocks and navigation buttons.

If you only want to use the ELD by yourself, you can continue in standalone mode.

3. Connect to your VTC with Discord
  1. Open your VTC Discord server.
  2. Use the linking command provided by your VTC, usually !link CODE.
  3. Return to OverWatch ELD after Discord confirms the link.
  4. The app should update from standalone mode to your connected VTC.
  5. You should now see VTC tools such as roster, dispatch, announcements, convoy, and fleet features.
Needed for VTC features
4. Set up ATS telemetry
  1. Install the ATS telemetry plugin required by your VTC or OverWatch ELD setup.
  2. Start American Truck Simulator.
  3. Load into your truck profile.
  4. Open OverWatch ELD.
  5. Check the dashboard for live truck data such as speed, fuel, location, damage, and odometer.
  6. If telemetry does not show, restart ATS and OverWatch ELD.

Telemetry powers live fleet tracking, load detection, convoy visibility, and dashboard stats.

Important for live data
5. Use the main dashboard
  • Duty Buttons: Change your current status such as Off Duty, Sleeper, Driving, or On Duty.
  • HOS Clocks: View live countdown clocks for drive time, shift time, cycle time, and break status.
  • Telemetry Status: Confirm ATS is connected and sending live data.
  • Companion Link: Use the phone/tablet dashboard when available.
6. Dispatch and load tracking
  • View assigned loads from dispatch.
  • Track pickup and delivery progress.
  • Send load activity to Discord if your VTC has it enabled.
  • Generate or review Bills of Lading when supported.
7. Logs and inspections
  • Review your duty history and log entries.
  • Certify logs when required.
  • Create vehicle inspections before or after trips.
  • Keep records organized for your VTC.
8. Companion app for phone or tablet
  1. Make sure your phone/tablet is on the same Wi-Fi network as your PC.
  2. Open the companion URL shown inside OverWatch ELD.
  3. Use the QR code button if available.
  4. Control duty status and view dashboard information from your mobile device.
9. Troubleshooting
  • No telemetry: Restart ATS and OverWatch ELD.
  • VTC not showing: Re-check your Discord link and make sure you are in the correct VTC server.
  • Companion not loading: Make sure both devices are on the same network.
  • Download issues: Use the downloads page again and install the latest version.
